Space Traffic Digital Twin · browser demo

↗ repository  ·  ↗ project write-up

Real orbits from the CelesTrak GP catalog, propagated with SGP4 in your browser — fetched live through the caching proxy at pierregathy.com (CORS-open), with the bundled snapshot in web/data/ as offline fallback. Serve this folder over HTTP (e.g. python -m http.server in web/) — module imports don't work from file://.
